# Promise Division Assignment
## Objective
The objective of this assignment is to create a JavaScript function that performs division using Promises. The function handles division by zero and returns either the result or an appropriate error message.

# Technologies Used
- HTML
- CSS
- JavaScript

# Features
- Uses JavaScript Promises
- Arrow function implementation
- Handles division by zero
- Displays results dynamically on screen
- Runs multiple test cases
- Uses `.then()` and `.catch()` for Promise handling

# How It Works
1. User clicks the "Run Test Cases" button
2. Multiple division test cases are executed
3. Promise resolves if division is valid
4. Promise rejects if divisor is zero
5. Results are displayed dynamically on screen
